Bienvenido al Ejercicio de Construcción de Contenido

Este sitio web es una herramienta educativa diseñada para estudiantes de séptimo semestre de Diseño Crossmedia. A través de este ejercicio práctico, desarrollarás habilidades en construcción de contenido, UX Writing y escritura para personas.

Duración: 2 horas
Modalidad: Colaborativo (GitHub)
Entrega: 30 noticias (3 por categoría)

¿Cómo realizar el ejercicio?

1

Lee los lineamientos

Revisa el documento LINEAMIENTOS_EJERCICIO.pdf para entender el tono y estilo de cada categoría.

2

Clona el repositorio

Descarga el proyecto desde GitHub y configura tu entorno de trabajo local.
🚀 Ir a github 🚀

3

Usa la plantilla

Duplica template-noticia.html para crear cada noticia siguiendo la estructura HTML.

4

Redacta contenido

Escribe 3 noticias por categoría aplicando principios de UX Writing y escritura para personas.
